# Upload with GitHub Desktop

Use this exact process:

1. Clone your repository in GitHub Desktop.
2. Click Repository → Show in Explorer.
3. Delete the old flattened files from that local folder.
4. Unzip this GitHub-ready repository.
5. Open the unzipped folder.
6. Copy everything inside it.
7. Paste it into the cloned GitHub folder.
8. Return to GitHub Desktop.
9. Commit message: Replace flattened upload with GitHub-ready app.
10. Click Commit to main.
11. Click Push origin.

The GitHub repository root must show:

- app
- components
- lib
- database
- public
- package.json
- next.config.ts
- tailwind.config.ts
- tsconfig.json
